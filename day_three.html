The Power Learn Project and Tech

The Power Learn Project is a private initiative that seeks to nurture talent in software development in Africa. The project aims to produce 1 million well-rounded developers in the continent. It has a learning platform through which the following courses are offered:

  • Python Programming
  • Web development
  • Database design
  • Dart and Flutter programming
  • Basic Computer skills
  • Entrepreneurship and soft skills
